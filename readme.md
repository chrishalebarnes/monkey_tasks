Monkey Tasks
============

## Getting Started
Install the depenencies

    bundle install

Migrate and seed the database

    rake db:migrate
    rake db:seed

Run the server

    rails s

Login as one of these cats with a password of 'password'

    monkey@monkeytasks.com
    tsavo@monkeytasks.com
    noche@monkeytasks.com

Run Tests

    rake spec
